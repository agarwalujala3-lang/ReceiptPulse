Place any licensed or user-supplied anime artwork here if you want exact visuals in the app.

Use these exact base names:

- `cast.png` / `cast.webp` / `cast.jpg`
- `bg-dark.png` / `bg-dark.webp` / `bg-dark.jpg`
- `bg-light.png` / `bg-light.webp` / `bg-light.jpg`
- `portal.png` / `portal.webp` / `portal.jpg`
- `splash.png` / `splash.webp` / `splash.jpg`

What each file does:

- `cast`: wide character group artwork shown near the lower stage area
- `bg-dark`: extra full-screen dark-mode background image
- `bg-light`: extra full-screen light-mode background image
- `portal`: tall cinematic side artwork
- `splash`: character art used in the startup branding animation

Best format:

- Transparent PNG or WEBP for `cast` and `splash`
- Wide landscape image for `bg-dark` and `bg-light`
- Tall portrait image for `portal`

If multiple versions exist, keep only the final one you want live.
