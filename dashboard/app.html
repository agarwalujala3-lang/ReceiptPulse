ReceiptPulse

Project Demo Booting dashboard. Portfolio Project Source Code

ReceiptPulse is a cloud-based portfolio project that uploads receipt files, extracts key fields with AWS, and stores them for a simple analytics dashboard.

AWS Textract Cognito Auth DynamoDB Storage

Portfolio Project

Upload receipts and review the extracted expense data.

The project connects S3, Lambda, Textract, DynamoDB, API Gateway, and Cognito. After sign-in, a user can upload a receipt and inspect the parsed result in the dashboard.
